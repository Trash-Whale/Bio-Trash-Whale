MPU6050Spark
============

MPU6050 library for Spark adapted from Sparkfun Arudino library by Paul Kourany, April 2014


Example MPU6050_raw.cpp is dependent on:

	MPU6050.h
	MPU6050.cpp
	I2Cdev.cpp
	I2Cdev.h
	
Example MPU6050_DMP6.cpp is dependent on:

	MPU6050.h
	MPU6050.cpp
	I2Cdev.cpp
	I2Cdev.h
	helper_3dmath.h
	MPU6050_6Axis_MotionApps20.h
